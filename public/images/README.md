# LP イラスト画像

このフォルダには、LPで使う画像を置けます。  
**画像を置かない場合**は、コード内の **inline SVG** が表示されます。

## 必要なファイル名（オプション）

| ファイル名 | 使用箇所 |
|-----------|----------|
| `hero.png` | ヒーロー（スマホと握手） |
| `confused.png` | 悩み：混乱 |
| `mother.png` | 悩み：育児中の通院 |
| `family.png` | 悩み：家族の不安 |
| `check.png` | 操作完了 |
| `recording.png` | 録音開始 |
| `security.png` | セキュリティの盾 |
| `doctor.png` | 安心の医師 |
| `patient.png` | 患者さん：AIにお任せ |
| `parent.png` | パパ・ママ：お子様の通院 |
| `summary.png` | AI要約機能 |
| `privacy.png` | 目的外利用なし |

推奨: 透過PNG、1:1 または 3:4、解像度は 400〜800px 程度。
