# LP イラスト画像

このフォルダ（`public/images/`）に画像を置くと、LPに表示されます。

## ご用意いただく画像（ベネフィットセクション）

次の2点をこのフォルダに配置してください。

| ファイル名 | 表示されるブロック |
|-----------|---------------------|
| **benefit-patient.png** | 「覚えること」はAIに任せて、先生の目を見て話す |
| **benefit-parent.png** | お子さんが泣いていても大丈夫。AIが聞いています |

- 画像がない間は、プレースホルダー（絵文字）が表示されます。
- 推奨: 正方形または4:3、幅 400〜600px 程度。角丸でトリミングしても問題ない構図にすると、LPの角丸枠に馴染みます。
