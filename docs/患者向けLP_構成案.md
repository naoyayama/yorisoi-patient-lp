# よりそい 患者向けLP 構成案

フォルダ内の実際のサービス内容（コンポーネント・ドキュメント）に基づいて組み直した構成案です。

---

## サービス概要（前提）

| 項目 | 内容 |
|------|------|
| **サービス名** | よりそい |
| **位置づけ** | AI診察メモサービス。あなたと家族に寄りそい。 |
| **提供価値** | 診察室の「わからない」を「安心」にかえる。録音→AI要約→LINEに届くをスマホ一台で。 |
| **特徴** | アプリ不要・LINEですぐ使える／医療従事者監修／完全無料（プロトタイプ版） |
| **対象** | 患者本人、パパ・ママ（お子様の通院）、ご家族（離れて暮らす親の通院） |

---

## 推奨セクション構成（現行LPとの対応付き）

以下の順序で、ユーザーが「共感→解決策→安心→行動」に自然に流れるようにしています。

### 2. こんな「もやもや」ありませんか？（課題・共感）

| 項目 | 内容 |

| **含める要素** | ・見出し：病院からの帰り道、こんな「もやもや」ありませんか？<br>・**患者様**：「先生の説明が難しくて覚えきれない…」→ 帰ってから「あれ、どうするんだっけ？」と不安になる。<br>・**パパ・ママ**：「子供が泣いていて聞き逃したかも…」→ ケアで手一杯。大事なポイントを後で再確認したい。<br>・**ご家族**：「親の病状を正確に把握したい…」→ 離れて暮らす親の通院内容を、漏れなく共有したい。<br>・締め＋CTA：その不安、「よりそい」が解決します。／無料で今すぐ解決する |
| **ビジュアル** | 各ペルソナ用イラスト（confused / mother / family） |

### 7. 使い方は、たったの3ステップ（利用フロー）

| 項目 | 内容 |
|------|------|
| **役割** | 申し込み〜利用開始の心理的ハードルを下げる |
| **現行コンポーネント** | `StepsSection.tsx` |
| **含める要素** | ・見出し：使い方は、たったの3ステップ。<br>・サブ：新しいアプリは不要。いつものLINEですぐに使えます。<br>・**STEP 1 開始**：LINEメニューの「診察メモ開始」をタップしてスマホを置くだけ。<br>・**STEP 2 診察**：いつも通り先生とお話ししてください。AIがしっかり聞いています。<br>・**STEP 3 完了**：診察が終わったら「完了」。数十秒でLINEに要約が届きます。<br>・注意：画面ロックの設定にのみご注意ください。 |
| **ビジュアル** | recording / hero流用（診察） / check |

### 8. 大切な情報は、厳重に守ります（信頼・セキュリティ）

| 項目 | 内容 |
|------|------|
| **役割** | 個人情報への不安を払拭し、医療連携の安心も伝える |
| **現行コンポーネント** | `TrustSection.tsx` |
| **含める要素** | ・見出し：大切な情報は、厳重に守ります。<br>・**匿名化処理**：データは個人が特定できないように加工して管理。プライバシー最優先。<br>・**目的外利用なし**：サービスの改善や研究目的以外（広告配信・外部販売など）には一切使用しない。<br>・**医療機関との連携で、さらに安心**：提携クリニックでは、AI要約を医師・スタッフが確認してから送信。誤った情報の心配がありません。※提携外の医療機関でもAI要約機能は利用可能。 |
