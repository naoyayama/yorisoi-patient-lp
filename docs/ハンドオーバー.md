## プロジェクト概要

- **名前**: よりそい（AI診察メモサービス）患者向けLP
- **技術**: React 19 + Vite 6 + TypeScript + Tailwind（CDN）
- **起動**: `npm install` → `npm run dev`（http://localhost:3000）
- **ビルド**: `npm run build` / プレビュー: `npm run preview`

## 直近で実施した対応

1. **ヒーロー背景イラスト**
   - デザインDNA（フラット・パステル・影なし・線クリーン）に合わせてヒーロー用背景イラストを生成。
   - 保存先: `public/images/hero-bg.png`
   - 内容: スマホ＋メモの線＋音声が流れ込む曲線。人物なし。左側余白多めでテキストオーバーレイ用。

2. **Hero セクションのレイアウト**
   - 背景に `hero-bg.png` を配置（Flexbox: `flex flex-col justify-center`、コンテンツは `flex flex-col items-center justify-center`）。
   - 背景は `absolute inset-0` + `bg-cover bg-center`、テキスト視認用に白〜surface のグラデーションオーバーレイを重ねている。
   - 参照: `components/Hero.tsx`

3. **ベネフィットの丸枠イラスト**
   - 円形で表示している patient.png / parent.png で、元の四角い枠（上下の横線）が見えないように `scale-125` を付与。
   - 親要素の `rounded-full overflow-hidden` でクリップ。
   - 参照: `components/BenefitsSection.tsx` の `<img className="... object-cover scale-125" />`

## イラスト・デザインルール

- **デザインDNA（patient.png / parent.png / hero-bg.png で統一）**
  - ミニマル・フラット・パステル（ティール・ミント・ピーチ・クリーム）、影・グラデなし。
  - 線は均一でやや手描き風。色は primary（#0F766E）/ primary-light（#CCFBF1）/ ink 等（`index.html` の tailwind.config 参照）。
- **配置**
  - ヒーロー背景: `public/images/hero-bg.png`
  - ベネフィット: `public/images/patient.png`, `public/images/parent.png`
- 詳細は `docs/イラスト_最小構成とデザイン案.md` および `public/images/イラスト仕様書.md` を参照。

---

## 注意点・今後の調整案

- **Hero の背景**
  - 背景の濃さは `Hero.tsx` の `opacity-[0.95]` とオーバーレイの `from-white/70` 等で調整可能。
- **ベネフィットの円**
  - 枠の線がまだ見える場合は `scale-125` を `scale-150` などに変更可能（BenefitsSection.tsx）。
- **画像がない場合**
  - BenefitsSection は画像読み込みエラー時にプレースホルダー表示。`public/images/` に patient.png / parent.png が存在することを前提としている。
